# Divyansh Srivastava - Portfolio (React)

## 🚀 Installation

```bash
npm install
npm start
```

## 📦 Build for Production

```bash
npm run build
```

## ✨ Features

- 🌙 Dark Mode Toggle with localStorage persistence
- ✨ Custom animated cursor
- 🎨 Particle network animation in hero
- 📊 Scroll progress bar
- 🎯 Smooth scroll reveal animations
- 📱 Fully responsive design
- ⚡ Pure CSS animations (no heavy libraries)

## 🛠️ Tech Stack

- React 18
- Pure CSS (no UI libraries)
- Playfair Display + DM Mono fonts
- Canvas API for particles

## 🎨 Customization

Edit colors in `App.css`:
- Light theme: `[data-theme="light"]` variables
- Dark theme: `[data-theme="dark"]` variables

## 📝 Notes

- All content is in respective component files
- Dark mode preference saved in localStorage
- Animations optimized for performance
- Mobile hamburger menu included
